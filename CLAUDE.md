Turn long-form podcast and YouTube content into viral short clips automatically.

## Features

- 📥 **YouTube Download** - Paste a URL, get the video
- 🎙️ **AI Transcription** - Full transcript of your content  
- ⚡ **Smart Highlights** - AI finds the best moments automatically
- 📱 **Auto Formatting** - 9:16 vertical format for TikTok/Reels/Shorts
- 💬 **Caption Generation** - Auto-generated captions
- 🎨 **Customizable** - Your brand, your style

## Quick Start

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/clipforge.git
cd clipforge

# Install dependencies
pip install -r requirements.txt

# Set up API keys
export GEMINI_API_KEY="your_key_here"

# Run
python app.py "https://youtube.com/watch?v=..."
```

## Tech Stack

- **Python** - Core language
- **yt-dlp** - YouTube/video downloading
- **FFmpeg** - Video processing
- **Gemini API** - AI analysis
- **Whisper** - Transcription
